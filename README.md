# RAG, math implementation
This is a RAG created from scratch, implemented algorithms without libraries using CUDA to reach more efficiency and created math model based in NestedLearning
This project implements a Retrieval-Augmented Generation (RAG) system from scratch. It includes custom CUDA implementations of the core algorithms (without relying 
on external libraries) to improve performance, as well as a mathematical model based on the Nested Learning philosophy to enhance memory and learning capabilities.
